== Requirements
* Java SE 8
* Apache Maven, Version 3.3.3
* Eclipse Java EE IDE for Web Developers, Version Neon Release (v4.6.3)

== Verwendete Bibliotheken
* Spring Boot
** Spring Boot Actuator
*** Dropwizard Metrics
* Spring AOP / AspectJ
* Spring Data JPA
** Hibernate
** H2
* Spring MVC
** Thymeleaf
** Bootstrap / jQuery

== Build mit Maven
Für die Konfiguration von Eclipse bitte das Eclipse-Plugin http://www.eclipse.org/m2e/[M2Eclipse, role="external", window="_blank"] verwenden.

[source,bash]
.Wertpapierverwaltung bauen
----
$ mvn clean install
----
